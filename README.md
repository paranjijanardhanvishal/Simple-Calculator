# Simple Interest Calculator

## Description
A modern, beginner-friendly web application to calculate simple interest and total amount based on the principal amount, annual interest rate, and time period. This project demonstrates foundational web development concepts including UI design, form handling, and basic DOM manipulation.

## Features
- Clean, responsive, and visually appealing user interface.
- Calculates Simple Interest accurately.
- Calculates Total Amount (Principal + Interest).
- Input validation to ensure proper values are entered.
- Real-time calculations with smooth micro-animations.

## Formula for Simple Interest
The simple interest formula is:
```
Simple Interest (SI) = (P × R × T) / 100
```
Where:
- **P** = Principal Amount (initial investment or loan amount)
- **R** = Annual Rate of Interest (in percentage)
- **T** = Time Period (in years)

## Example Calculation
If you invest **$10,000** (P) at an annual interest rate of **5%** (R) for **3 years** (T):
- **SI** = (10000 × 5 × 3) / 100 = **$1,500**
- **Total Amount** = Principal + SI = 10000 + 1500 = **$11,500**

## Technologies Used
- **HTML5**: For structuring the user interface.
- **CSS3 (Vanilla)**: For styling, layout, and smooth animations.
- **JavaScript (ES6+)**: For calculations and DOM manipulation.

## How to Run

   ```bash

   ```
2. Navigate to the project directory:
   ```bash
   cd simple-interest-calculator
   ```
3. Open `index.html` in your web browser. No server setup is required!
